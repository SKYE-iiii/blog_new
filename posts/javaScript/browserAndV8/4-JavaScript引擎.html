            # （一）为什么需要 JavaScript 引擎

            1. JavaScript 是一种属于网络的高级脚本语言，高级的编程语言需要转成最终的机器指令来执行
            2. 我们编写的 JavaScript，无论交给 浏览器 或 Node 执行 ， 最后都是需要被 CPU 执行（ ps：CPU（中央处理器）：是计算机中负责读取指令，对指令译码并执行指令的核心部件）
            3. CPU 只认识自己的指令集，即机器语言，才能被 CPU 执行
            4. 使用 JavaScript 引擎将 JavaScript 代码 翻译成 CPU 指令 来执行
            详情

            个人理解 ： JavaScript 代码最终是由 CPU 执行的，而 CPU 只认识自己的指令集，使用 JavaScript 引擎可以将 JavaScript 代码翻译成 CPU 认识的指令集来执行

            # （二）常见的 JavaScript 引擎

            1. SpiderMoney ：第一款 JavaScript 引擎，由 Brendan Eich 开发 （JS 作者）
            2. Chakra ： 微软开发
            3. JavaScriptCore ：Webkit 中的 JavaScript 引擎，Apple 开发
            4. V8 ： Chrome 开发
